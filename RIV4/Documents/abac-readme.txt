1. Lancia la macchina virtuale
2. Avvia "Advanced BAT to EXE Converter PRO" (ABEC)
3. Apri il file setup.bat dal desktop
4. Apporta le eventuali modifiche
5. Quando tutto è pronto fai una delle seguenti:
	CTRL+B										- oppure - 
	clicca sul secondo pulsante della "toolbar"	- oppure - 
	vai nel File » Build EXE
6. Troverai già inserite le informazioni corrette, ma puoi modificarle.
	In particolare è importante che sia selezionata la casella 
		"Include Administrator Manifest".
	È opportuno selezionare la casella "Start Invisible"

Ho modificato l'icona di RuralInvest e l'ho salvata sul Desktop e ABEC è già
configurato per usarla.
 
Generalmente è sufficiente cliccare sul pulsante "Build EXE" e il nuovo 
setup.exe sarà sul desktop.
