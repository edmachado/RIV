RuralInvest Quick-Start

Which version to install

Before proceeding it is important to install the correct version of RuralInvest. There are 2 versions:

  1. Administrator:
    This version is intended for the administrator of a collection of RuralInvest installations. The full configuration of the system may be modified including the Language, Field Offices, Categories, Beneficiaries, Settings (Discount rate, Maximum project duration, Currency, Exchange rate, Locations, Loan parameters, Quick links etc.).

    NOTE: This version should NOT be generally available

  2. User:
    This version is intended for end users. With the exception of defining new users, the configuration of the system may NOT be modified. The configuration of the installation is accomplished by importing predefined settings provided by an administrator. To download the installer ask your FAO referrer.

    NOTE: This version can be freely distributed

Both versions support exporting and importing configuration settings which are fully interchangeable and compatible.

Directly after installation both versions will be slightly slower than may be expected. After a period of initial use (once each page has been visited) the installation will reach optimal performance.

Using the Administrator version for the first time

During the installation you will be prompted to choose a username and a password, you can then login to the application.

NOTE: Remember your username and password, as there are no way to recover your password. If you forget your password you will need to reinstall.

On completion you will be prompted to log in as the newly created user. Once you have entered the system as the newly created user you will be prompted to complete the configuration for the installation.

For the administrator version, the configuration can be either set manually or imported from a settings file. [See riv_import_settings.gif]

As an administrator you can share the configuration settings for use in user installations using Configuration->Settings->Export [See riv_export_settings.gif]

Using the User version for the first time

During the installation you will be prompted to choose a username and a password, you can then login to the application.

NOTE: Remember your username and password, as there are no way to recover your password. If you forget your password you will need to reinstall.
