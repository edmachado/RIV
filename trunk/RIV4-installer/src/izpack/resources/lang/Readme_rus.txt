RuralInvest
Version: $app-version

Quick Start
===========

RuralInvest is started by selecting "Start RuralInvest" from the "Start" menu.  
A web browser window will be opened displaying RuralInvest.


User Guide
============

The User Guide to the RuralInvest software is available on the installation CD.  
This includes information on the installation procedure and on use of the software.

Uninstall
=========

RuralInvest may be uninstalled selecting "Uninstall RuralInvest" from the "Start" menu.
