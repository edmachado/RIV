Démarrage rapide

Pour démarrer RuralInvest, sélectionner "RuralInvest" depuis la barre de démarrage "Démarrer" ("Start"), puis "Tous les programmes". Une page d'accès au logiciel s'ouvira alors dans votre navigateur internet.

Guide de l'utilisateur

Le Guide de l'utilisateur du logiciel RuralInvest est inclus dans le CD d'installation. Il est également accessible via le Menu principal du logiciel sous la section "Aide" puis "Manuels de l'utilisateur". Ce guide comprend notamment les informations concernant les procédures d'installation/mise à jour et d'utilisation détaillée du logiciel.

Désinstallation

RuralInvest peut être désinstallé en sélectionnant "Désintaller RuralInvest" depuis le menu de démarrage de de votre ordinateur sous la section "Tous les programmes" puis "RuralInvest".
