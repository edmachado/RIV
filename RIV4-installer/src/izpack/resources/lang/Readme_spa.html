Inicio Rápido

RuralInvest es iniciado al seleccionar "Inicie RuralInvest" del menú de "Inicio". Se abrirá una ventana de navegador web mostrando RuralInvest.

Guía de usuario

La Guía del Usuario para el software de RuralInvest es disponible en el CD de instalación y en el software en el menú "Ayuda -> Manuales del Usuario". Este incluye información sobre el proceso de instalación y sobre el uso del software.

Desinstalar

RuralInvest puede ser desinstalado seleccionando "Desinstalar RuralInvest" del menú de "Inicio".
